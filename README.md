# Customer Segmentation using Machine Learning

## Overview
This project uses K-Means Clustering to segment customers based on their annual income and spending behavior.

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Features
- Exploratory Data Analysis (EDA)
- Elbow Method
- K-Means Clustering
- Customer Group Identification
- Business Insights

## Results
Identified 5 customer segments:
1. Premium Customers
2. Wealthy Conservative Customers
3. Young Enthusiastic Customers
4. Budget Customers
5. Average Customers

## Dataset
Mall Customers Dataset
